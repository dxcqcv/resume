# Introduce 
written resume online with webpack, jade and stylus.

# Usage
* `npm install` for install all package
* `gulp` just for develop
* `NODE_ENV=production gulp deploy` for make whole webpage to gh 

## publish
1. delete dist
2. `npm run dev`
3. `npm run build`
